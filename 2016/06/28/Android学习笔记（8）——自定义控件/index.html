Android学习笔记(8)——自定义控件

  • Posted on: 2016-06-28
  • By: Jxust_CGS
  • With:

如何实现一个自定义控件

  • 定义自定义属性
  • 在layout文件中放置自定义控件
  • 新建一个类CustomView继承view
  • 实现其中的方法
    • 构造方法
      • CustomView（Context context）{}//在java代码中实例化这个对象是调用
      • CustomView（Context context,AttributeSet attrs）{}//通过xml中实例化对象时调用
      • CustomView（Context context,AttributeSet attrs,int defStyle）{}//通过xml实例化对象并应用默认熟悉（如：在theme中定义的属性）
    • onMeasure（int widthMeasureSpec,int heightMeasureSpec）{}//此方法非必须，通过重写此方法来定义控件的大小
    • onDraw（Canvas canvas）{}//通过实现此方法来绘制自定义组件

定义自定义控件属性

定义属性的作用

定义自定义控件属性的作用是在layout文件中放置自定义组件时为自定义控件初始化参数（如同默认控件中有android：layout_width=“warp_context”）。

如何定义属性？

在项目中的values文件夹下新建attr.xml并进行编辑。 

<?xml version="1.0" encoding="utf-8"?>
<resources>
    <!-- 定义自定义组件中的参数 -->
    <attr name="titleText" format="string"/>

    <declare-styleable name="CustomTitleView">//定义组件"CustomTitleView"的属性
        <attr name="titleText"></attr>
        <attr name="titleTextColor" format="color"></attr>
        <attr name="titleTextSize" format="dimension"></attr>
    </declare-styleable>
</resources>

常见属性

属性类型 format 获取方式 应用场景
int integer getInteger（attr，defultInt） 获取数字
string string getString（attr） 获取字符串
int color getColor（attr，defultCol） 获取颜色
int dimension getDimensionPixelSize(attr,(int) TypedValue.applyDimension(TypedValue.COMPLEX_UNIT_SP, 16, getResources().getDisplayMetrics())); 获取字体大小
bitmap reference BitmapFactory.decodeResource(getResources(), array.getResourceId(attr, 0)); 获取图片资源

另附枚举类型 

<attr name="image_align">
    <enum name="center" value="0"/>
    <enum name="left" value="1"/>
    <enum name="right" value="2"/>
</attr>

在layout文件中放置自定义控件

方法和默认控件一样有一点不同的是需要配置自定义属性就需要定义属性命名控件（即注册之前定义的属性）。

<RelativeLayout xmlns:android="http://schemas.android.com/apk/res/android"
    xmlns:tools="http://schemas.android.com/tools"
    xmlns:custom="http://schemas.android.com/apk/res/com.example.privateview（项目包名）" //注册自定义属性命名空间
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    tools:context="${relativePackage}.${activityClass}" >
    <com.example.privateview.CustomTitleView//自定义控件的类名
        android:id="@+id/customTitleView1"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:padding="10pt"
        custom:titleText="2345"//配置自定义属性
        custom:titleTextColor="#ffffff"
        custom:titleTextSize="90sp" />
</RelativeLayout>

构造方法

构造方法的作用就是实例化此对象，也就是要对此对象进行初始化，所以此方法中需要完成的工作就是对此自定义的控件中定义的各个属性进行赋值。
赋值需要的参数需要读取到在layout文件中配置的属性，我们通过TypedArray获取 

public CustomTitleView(Context context) {
    this(context,null);
}
public CustomTitleView(Context context,AttributeSet attrs) {
    this(context,attrs,0);
}
public CustomTitleView(Context context,AttributeSet attrs,int defStyle) {
        super(context, attrs, defStyle);
        TypedArray array=context.getTheme().obtainStyledAttributes(attrs,R.styleable.CustomTitleView,defStyle,0);//获取配置的参数
        int n=array.getIndexCount();
        for(int i=0;i<n;i++){
            int attr=array.getIndex(i);
            switch (attr) {
            case R.styleable.CustomTitleView_titleText:
                mTitleText=array.getString(attr);
                break;
            case R.styleable.CustomTitleView_titleTextColor:
                mTitleTextColor=array.getColor(attr, Color.BLACK);
                break;
            case R.styleable.CustomTitleView_titleTextSize:
                mTitleTextSize=array.getDimensionPixelSize(attr, (int) TypedValue.applyDimension(TypedValue.COMPLEX_UNIT_SP, 16, getResources().getDisplayMetrics()));
                break;
            }
        }
        array.recycle();//读取完毕后一定要记得回收
        mPaint_background=new Paint();//画笔
        mPaint_background.setColor(Color.GRAY);
        mPaint_word=new Paint();
        mPaint_word.setTextSize(mTitleTextSize);
        mPaint_word.setColor(mTitleTextColor);
        mBound=new Rect();//矩形（文本框）
        mPaint_word.getTextBounds(mTitleText, 0, mTitleText.length(), mBound);
        this.setOnClickListener(new OnClickListener() {
            @Override
            public void onClick(View v) {
                mTitleText=randomText();
                postInvalidate();//刷新控件，重新绘制控件
            }
        });
    }

onMeasure

用于向ViewGroup反馈组件的大小所占空间，如果不写此方法控件会占据界面所有剩余空间，无论是match_parent还是warp_content. 

/**
 * MeasureSpec的specMode,一共三种类型：
 * EXACTLY：一般是设置了明确的值或者是MATCH_PARENT
 * AT_MOST：表示子布局限制在一个最大值内，一般为WARP_CONTENT
 * UNSPECIFIED：表示子布局想要多大就多大，很少使用
 */
@Override
protected void onMeasure(int widthMeasureSpec,int heightMeasureSpec){
    super.onMeasure(widthMeasureSpec, heightMeasureSpec);
    int widthMode=MeasureSpec.getMode(widthMeasureSpec);
    int heightMode=MeasureSpec.getMode(heightMeasureSpec);
    int widthSize=MeasureSpec.getSize(widthMeasureSpec);
    int heightSize=MeasureSpec.getSize(heightMeasureSpec);
    int width;
    int height;
    if(widthMode==MeasureSpec.EXACTLY){
        width=widthSize;
    }else{
        width=getPaddingLeft()+mBound.width()+getPaddingRight();
    }
    if(heightMode==MeasureSpec.EXACTLY){
        height=heightSize;
    }else{
        height=getPaddingTop()+mBound.height()+getPaddingBottom();
    }
    setMeasuredDimension(width, height);
}

onDraw

在界面中绘制出控件

@Override
protected void onDraw(Canvas canvas){
    super.onDraw(canvas);
    canvas.drawRect(0, 0,getMeasuredWidth(),getMeasuredHeight(),mPaint_background);//在画布上画下背景矩形
    canvas.drawText(mTitleText, getWidth()/2-mBound.width()/2, getHeight()/2+mBound.height()/2,mPaint_word);//在画布上画下文本框
}
