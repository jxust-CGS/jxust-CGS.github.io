使用groumble.js给页面添加泡泡提示框

发表于:2015-04-21，更新于:2015-05-05，By Jxust_CGS
大纲
  1. 1. 使用的方式
  2. 2. 配合jquery可以使web页面更加友好
  3. 3. 获取更多

使用的方式

首先是需要下载到groumble的css和js文件（可以访问官方github）
在页面中正确链接css和js文件
绑定页面的DOM元素或事件调用js
例如在密码是输入框选中是弹出提示框 

<script>  
    $("#userPassword").focus(function(){  
        $(this).grumble(  
            {  
                text: '复杂度高的密码更加安全哟！',//泡泡中的文本
                angle: 85,//上偏右角度值
                type: 'alt-', //泡泡的外形类型
                distance: 50,//离dom中心的距离
                showAfter: 0,//触发后-ms后出现
                hideAfter: 2000,//出现后-ms后消失，不消失为false
                hasHideButton: false,//有无消失按钮
                buttonHideText: 'Pop!'//点击后消失
            }
        );
    });
</script>
